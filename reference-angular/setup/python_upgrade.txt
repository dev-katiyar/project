INSTALL APACHE: 
    - Download Apache (2.4.52 Win64VS16) https://httpd.apache.org/download.cgi
    - Unzip it 
    - Place it in the right folder e.g. software/Apache24

INSTALL PYTHON: 
    - Download Python (3.9.2 x64)  https://www.python.org/downloads/
    - Install using the installer 
    - Set Python 3.9 as default Python for machine: This PC → Right-click → Click on Properties → 
        Advanced System Settings -> System Properties -> Advanced Tab -> Environment Variables -> 
        System variables (lower half) -> double-click on the Path (to edit the Path) -> 
        Move/Add Python3 Path ahead in the list. -> Restart the Command Prompt -> check by typig 'python'.


Note: Make sure that the python version you're using has the same architecture (32/64 bit) as your Apache version.

WSGI INSTALL AND SETUP: 
    - Install mod-wsgi package:  https://dev.to/willmvs/flask-deployment-on-windows-139b
        - Point MOD_WSGI_APACHE_ROOTDIR to your apache installation (default is C:/Apache24). Use forward slashes.
            - set MOD_WSGI_APACHE_ROOTDIR e.g. execute command ' set "MOD_WSGI_APACHE_ROOTDIR=C:/softwares/Apache24" '
        - INSTALL MICROSOFT VISUAL C++ Build Tools (follow steps as on dev.to link above): https://visualstudio.microsoft.com/visual-cpp-build-tools/
        - Execute 'pip install mod-wsgi'

    - Get module configuration:
        - Executre 'mod_wsgi-express module-config'
        - Copy the output of the previous command into your Apache's httpd.conf.


CONFIGURE APACHE: 
    - edit httpd.conf file (right now, copy relevant parts from the already running system)
    - Virtual Host tag (Direcgtory tag inside it)


FILES
    - Copy front end code in the target directory 
    - Copy backend files in the directory

INSTALL PYTHON DEPENDENCIES
    - requirements.txt file 
    - pip install requirements_3.9.txt
