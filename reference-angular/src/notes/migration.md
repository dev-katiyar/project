Migration to Angular 11
=======================

node : compiles ts to js (npm and CLI depends on node)
npm  : node package manager (installs new libs/packages, it depends on node so first update node to higher version)
angular CLI : ng new , ng g , ng serve , ng build

ng --version (shows Angular CLI and Node version)
node -v (to check just the node version)
v15.14.0 (latest node version)

Steps for angualr11:
1). Create new directory
2). Install specific CLI version : npm install @angular/cli@~11.2.13
3). Install primeng version compatible with Angular11 i.e
    npm install primeng@11.4.2
    npm install primeicons

OR run github downloaded angular project from your computer.

github link : https://github.com/priyankajalal/Angular11

1).GO into your downloaded folder.

2).Open command prompt and write command :
	npm install
    ng serve (after completing the installation)

 Your project automatically open in your browser.
